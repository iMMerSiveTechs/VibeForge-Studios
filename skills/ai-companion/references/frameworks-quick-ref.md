# Dan Martell Frameworks — Quick Reference

## Decision Tree: Which Framework?

```
User is stuck on WHAT to build → PTCF + DRAG
User is stuck on HOW to build → Hill/Gym/Fool
User is doing too much → Buyback Principle + Energy Audit
User needs to delegate → Replacement Ladder + 10-80-10
User needs to decide → 1:3:1 Rule
User needs to position/sell → Transformation Selling
User needs growth strategy → Growth Flywheel
```

## One-Liner Summaries

| Framework | One-Liner |
|-----------|-----------|
| PTCF | Frame the real problem before solving it |
| DRAG | Map the customer's journey from pain to dream |
| Hill/Gym/Fool | Diagnose: wrong plan, wrong skills, or wrong mindset? |
| Buyback | Stop doing $10/hr work |
| Replacement Ladder | Replace Admin first, Sales last |
| 1:3:1 | 1 problem, 3 options, 1 recommendation |
| Energy Audit | Delegate what drains you |
| 10-80-10 | Own the first 10% and last 10%, delegate the middle |
| Transformation | Sell the identity shift, not features |
| Flywheel | Content → Leads → Sales → Delivery → Referrals |
